# Preserve Battery Health Quick Setting for GNOME

This allows you to seamlessly support battery health preservation features on the Gnome desktop environment. It provides a quick setting toggle to preserve battery health by enabling or disabling the charge threshold, enhancing battery longevity directly from Gnome Shell.
